== Description ==

Gavii gallery allows users to create a unique and colorful sidebar. It displays the entire contents on an album in a beautiful tiled widget.

If a user clicks the selected image they will be directed via a new window so they can view it full size. 


== Installation ==

1. Upload `gavii-gallery.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the `gavii gallery` widget into your sidebar.
4. Sign up at http://gavii.com/register.php
5. create an album and copy your album key.
6. enter your album key and start uploading to gavii
7. If you have made use of the widget images will automatically be added to your sidebar.

== Frequently asked questions ==

I've added the widget, what is the "Album Key"?

The album key is supplied once you create an album at www.gavii.com. Each album key is unique and will only work with that album.


I've installed the widget but i get a message saying "Request method 'GET' not supported"?

This is an issue with your howing provider, we strngly suggest you ask your hosting support team.
